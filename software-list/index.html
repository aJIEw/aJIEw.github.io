我的日常使用工具清单

发表于 2020-01-14 | 分类于 随笔 | 阅读数
字数统计: 3.4k | 阅读时长 ≈ 13

工欲善其事，必先利其器。拥有一个好的工具集可以极大地提高我们的生产力，因此在这里整理一下我觉得必备的工具，作为记录和分享。

我知道某些工具可能有不少替代品，但是出于个人喜好、使用习惯等原因一直没有动力切换。但是，如果你有特别好的推荐，也欢迎留言分享。

我会根据平台和功能来整理，如果某个工具支持多个平台，那么就不在另一平台下重复列出了。链接我尽量使用该软件的官方网站，如果没有官网，则用该软件所在平台的官方发布渠道的链接来代替。

### Chrome

Chrome 应该是我日常使用频率最高的一个工具了，如何把 Chrome 打造得更好用是我一直在探索的事情，所以这里把它单独拎出来。

Chrome 目前虽然已经更新到 80+ 了，但是有一些核心的功能却还是缺失的，所以只能通过安装 Extension 来解决。

Flags

一些实验性的功能 (Eexperimental Features) 默认可能是关闭的，只能通过 chrome://flags/ 手动打开。下面是我使用最多的一些实验性功能，有些功能未来可能会在更新后默认开启，而有些可能会被移除。请谨慎使用，切勿依赖。

通用
名称 功能介绍
Parallel downloading 多线程下载，可提升多个文件时的下载速度
Sync Clipboard Service 剪贴板同步服务（需要开启下面两个）
Enables the remote copy feature to receive messages 剪贴板共享发送端
Enable shared clipboard feature signals to be handled 剪贴板共享接收端
Experimental QUIC protocol 使用基于 UDP 的 QUIC 协议，可以提升连接速度、降低延迟，看视频时体验更流畅
Back-forward cache 前进后退缓存，提升页面加载速度
桌面端
名称 功能介绍
Tab Groups 标签页分组控制
Global Media Controls 媒体播放全局控制，可以在工具栏中统一控制标签页中音乐或视频的播放
移动端
名称 功能介绍
Chrome Duet 操作栏位于底部，在大屏手机上特别实用（重启浏览器后如果没效果可以尝试从最近列表中杀死应用再重新进入）
Tab Grid Layout 标签页网格排列

另外，通过 chrome://extensions/shortcuts 给一些常用的扩展设置快捷键也能带来不少便利，但是小心不要和 vimium 的快捷键冲突了。
