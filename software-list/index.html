我的日常使用工具清单

2020-01-14
随笔
Notes, Softwares, Tools

工欲善其事，必先利其器。拥有一个好的工具集可以极大地提高我们的生产力，因此，在这里整理一下我觉得必备的工具，作为记录和分享。

我知道某些工具可能有不少替代品，但是出于个人喜好、使用习惯等原因一直没有切换，所以，如果你觉得有更好的推荐，欢迎留言分享。

我会根据平台和功能来整理，如果某个工具支持多个平台，那么就不再在另一平台下重复列出了。链接我尽量使用该软件的官方网站，如果没有官网，则用该软件所在平台的主流发布渠道的链接替代。

插件介绍

这里介绍一下部分插件的使用。

Bypass Paywalls

用于绕过付费墙的限制。由于版权问题，只能通过 github 下载。

Color by Fardos - Color Picker

用于网页取色。可以在 shortcuts 中设置快捷键：

  • Pick colors from site: shift + option + C
QR Code Generator

用于生成当前网站或者任意内容的二维码。可以在 shortcuts 中设置快捷键：

  • Activate the extension: option + Q
Octo Mate

用于提升 GitHub 的使用体验，比如添加侧边栏等。该插件目前无法通过 Chrome Web Store 下载，我上传了一个备份，可以通过 Developer mode 加载该插件。

Reader View

进入阅读模式，使网站内容更易阅读。可以在 shortcuts 中设置快捷键：

  • Toggle the Reader View: shift + option + R
Rediretor

用于重定向网页，通过正则匹配网址。比如，将移动端 Wikipedia 网址替换为桌面端网址：

  • ^https://(.?).m.wikipedia.org/(.) -> https://$1.wikipedia.org/$2
Super Simple Highlighter

用于高亮网页内容。可以在 shortcuts 中设置快捷键：

  • highlight 1: shift + option + A
uBlacklist

用于屏蔽网站。可以添加订阅 Google-Chinese-Results-Blocklist 的屏蔽规则：

  • 精确匹配：https://raw.githubusercontent.com/cobaltdisco/Google-Chinese-Results-Blocklist/master/uBlacklist_subscription.txt
  • 模糊匹配：https://raw.githubusercontent.com/cobaltdisco/Google-Chinese-Results-Blocklist/master/uBlacklist_match_patterns.txt

Flags

一些实验性的功能 (Eexperimental Features) 默认可能是关闭的，只能通过 chrome://flags/ 手动打开。下面是我使用最多的一些实验性功能，有些功能未来可能会在更新后默认开启，而有些可能会被移除。请谨慎使用，切勿依赖。

通用
名称 功能介绍
Parallel downloading 多线程下载，可提升多个文件时的下载速度
Enables the remote copy feature to receive messages 剪贴板共享发送端
Enable shared clipboard feature signals to be handled 剪贴板共享接收端
Experimental QUIC protocol 使用基于 UDP 的 QUIC 协议，可以提升连接速度、降低延迟，看视频时体验更流畅
Back-forward cache 前进后退缓存，提升页面加载速度
注：最新版本的 Chrome 中已去除剪贴板分享，如需继续使用需要安装老版本，比如 v90.0.4430，并且禁用更新

  • 豆瓣FM - 私人电台推荐的歌很合我口味
  • 网易云音乐 v4.3.1 - Play Store 下架前的最后一个版本，永不升级的应用之一
  • Google Podcasts - 目前用得最多的播客应用
  • Radio Garden - 可以收听世界各地的电台节目
  • doubleTwtist - 本地音乐播放器，极客专用
  • VLC - 本地视频播放器
  • TED - 用于看最新的 TED 视频

游戏类

  • Call of Duty - 刚出的时候追过两个 Season，还不错，没玩过的推荐尝试下
  • Asphalt 9 - 最爱的竞速类手游 No.1
  • Asphalt Xtreme - 最爱的竞速类手游 No.2
  • EVE Echoes - 社交聊天软件搬到手机端啦~
  • Chilly Snow - 简单但却考验耐心
  • Avicii | Gravity HD - 音乐类手游
  • 狼人杀 - 玩了一年依旧玩不转，需要你学会猜测他人的心理，适合练习逻辑推理和临场表达能力
  • Toontastic - 用于陪学龄前小朋友学讲故事的工具

工具类

  • Shortcut Maker - 创建应用启动器，查看应用的 Activity
  • Current Activity - 查看当前显示的 Activity 名
  • Display Info - 屏幕基本信息
  • Device Info HW - 查看手机硬件信息
  • LADB - 本地 adb shell
  • Clipper - 通过 adb 和系统剪贴板进行交互，比如获取和设置剪贴板内容

其他

  • Kickstarter - 寻找创意、灵感
  • Expeditions - AR & VR
  • Daydream - VR（需要手机支持）

  • ASL Signs - 学习美式手语
