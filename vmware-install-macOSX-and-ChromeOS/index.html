VMware 下安装 macOS 和 Chrome OS

发表于 2017-04-09 | 分类于 小技巧 | 阅读数
字数统计: 445 | 阅读时长 ≈ 1

最近闲来无事，突然想用 VMWare 安装 macOS 和 Chrome OS 玩玩。虽然网上教程有很多，但是每次去搜总是花费掉不少时间，所以记录下来以备不时之需。

Prerequisite

操作系统：Windows 10, 64-bit (Build 17134) 10.0.17134

VMware 版本：VMware® Workstation 12 Pro (Build 3272444) 12.1.0

安装 Mac OS

网上这部分教程很多，基本步骤就是先下载并运行 Mac OS X Unlocker for VMware，这样 VMWare 就能够识别 Mac OS 了，之后新建虚拟机并安装就可以了。

如果遇到了“不可恢复错误：(vcpu-0)”，只要打开 Mac OS 虚拟机根目录下的 .vmx 文件，并添加一行“smc.version = 0”，然后重启就可以解决。

如果遇到其他提示错误，一般只要关闭虚拟机，并删除根目录下的 .lck 文件夹并重启虚拟机就可以解决。

安装 Chrome OS

一开始花了不少时间在找 Chrome OS 的镜像文件上，但其实 VMWare 同样不能识别 Chrome OS，所以我们需要的是直接包含了 Chrome OS 的虚拟硬盘 (.vmdk) 文件，可以从 neverware 网站上获取，这里我使用的是一个非官方版本，下载解压后运行 .ovf 文件即可导入 Chrome OS 到 VMWare 中了。

当然，Chrome OS 必须在联网后才能愉快使用，毕竟一切程序都是从网络上获取的，不联网的话只能以访客身份进入。
