基础设置

最佳屏幕显示

为了让 Retina 屏幕发挥出最大的作用，让文字显示达到最佳效果，我们需要关闭 font smoothing[1] 和修改屏幕分辨率[2]。参考文章：Time to upgrade your monitor by @Nikita。

对应的设置方法：

1）打开 System Preferences > General，解选 “Use font smoothing when available”；如果是 Big Sur 之后的系统，则需要通过在命令行中输入 defaults -currentHost write -g AppleFontSmoothing -int 0 并重启。

2）打开 System Preferences > Displays，Display 中 Resolution 选择 Scaled，然后选择 “Looks like 1536x960”（如果默认就是这个选项，那么就不需要修改了）。
